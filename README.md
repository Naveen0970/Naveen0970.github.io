# AEM Developer Portfolio

A fast, self-contained portfolio site — no frameworks, no build step, free to host on GitHub Pages.

## Files
- `index.html` — homepage (hero, live component playground, skills, experience timeline, work, contact)
- `case-study.html` — flagship project write-up (problem, architecture diagram, code, metrics)
- `assets/style.css` — all styling (dark theme + light toggle)
- `assets/script.js` — theme toggle, scroll animations, the playground, metric counters

## 1. Make it yours
Open the two `.html` files and replace everything in `[SQUARE BRACKETS]` with your details:
name, role, links, skills, experience, and the case study content. Delete any section you
don't need (for example, the second work card if you only have one project).

Add your resume as a PDF in this folder and point the `[link-to-resume.pdf]` links to it.

## 2. Put it online (free, GitHub only)
1. Create a new GitHub repository named exactly: `yourusername.github.io`
2. Upload all files **keeping the folder structure** (all files sit at the top level (no subfolder)).
3. In the repo: **Settings → Pages → Source → Deploy from branch → `main` / root → Save**
4. Wait ~1 minute. Your site is live at: `https://yourusername.github.io`

## 3. Add it to LinkedIn
Profile → Contact info → Website → paste your URL. Also drop the link in your headline
or About section so recruiters see it immediately.

## Tips
- Keep skill chips honest — recruiters often verify.
- One deep case study beats five thin project cards.
- Want a custom domain later? Buy one (~$10/yr) and add a `CNAME` file. Optional.
